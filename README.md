# Image Morphing
## Dependencies
1. `pip install -r requirements.txt`
2. `python download_shape_predictor.py`
## Build
* `python build.py`  
## Usage
### Run without building:
1. `cd src/`
2. `python ImageMorphGUITK.py`  
### Run after building:
1. `cd ImageMorphGUITK/`
2. `ImageMorphGUITK.exe`
